# Contributing to bunny-api-proxy

Thank you for your interest in this project.

## Code Contributions

**This project is not accepting external code contributions (pull requests) at this time.**

All code is written and maintained by the project author. This approach ensures:
- Consistent code quality and style
- Full control over the commercial licensing model
- Clear intellectual property rights

## How You Can Help

While we're not accepting code contributions, there are other ways to contribute:

### Bug Reports

If you find a bug, please open a GitHub Issue with:
- Clear description of the problem
- Steps to reproduce
- Expected vs actual behavior
- Environment details (OS, Go version, etc.)

### Feature Requests

Feature requests are welcome as GitHub Issues. Please include:
- Use case description
- Why this feature would be valuable
- Any implementation ideas (optional)

### Documentation

Suggestions for improving documentation are appreciated. Please open an Issue describing:
- What's unclear or missing
- What documentation would be helpful

## Questions and Discussions

For questions about:
- **Usage**: Open a GitHub Issue with the "question" label
- **Commercial licensing**: Open an Issue or contact the maintainer
- **Security issues**: See SECURITY.md (if available) or open a private security advisory
